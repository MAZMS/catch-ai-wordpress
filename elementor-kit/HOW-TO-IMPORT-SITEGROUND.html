Catch AI — Import Guide for SiteGround

Put this page on your SiteGround-hosted WordPress site with Elementor (free — no Pro required).

Before you start

  • An active SiteGround plan with WordPress installed. (Not installed yet? Site Tools → WordPress → Install & Manage.)
  • Recommended: test on a staging copy first — Site Tools → WordPress → Staging → Create Staging Copy (GrowBig / GoGeek plans), then Push to Live when happy.
  • Have this kit's catch-ai-elementor-template.json and the assets/ images ready on your computer.

Import in 6 steps

  1. Log into WordPress: go to https://YOURDOMAIN.com/wp-admin and sign in. (This is your WordPress login — separate from your SiteGround Site Tools login.)
  2. Install Elementor (free): Plugins → Add New → search Elementor → Install Now → Activate.
  3. Import the design: Templates → Saved Templates → Import Templates → upload catch-ai-template-IMPORT.zip (or the catch-ai-elementor-template.json file) → Import Now. Don't upload the whole kit zip — just this template file.
  4. Create the page: Pages → Add New → give it a title → Edit with Elementor → open the folder / library icon → My Templates → find Catch AI — Home → Insert.
  5. Set full-width layout: Page Settings (gear, bottom-left) → Page Layout → Elementor Canvas → Publish. (Removes the theme header/footer — the design ships its own.)
  6. Global colors & fonts: Elementor → Site Settings → Global Colors / Global Fonts (table below).
⚠ Clear SiteGround cache after publishing. SiteGround caches aggressively, so a new/updated page can look old or unstyled until you flush it:
  • SG Optimizer plugin (in wp-admin) → Caching → Purge SG Cache, and
  • Site Tools → Speed → Caching → Dynamic Cache → Flush Cache.
  • Still looks broken? In SG Optimizer → Frontend Optimization, temporarily turn off "Minify CSS/JS" and "Combine CSS Files", then flush again.
No extra CSS setup: the sticky header and the hero's layered phone/photo are driven by a small CSS block that travels inside the template (an HTML widget in the hero), so a fresh import carries it automatically.
Make it the homepage (optional): Settings → Reading → A static page → Homepage → select this page. If your active theme is a block/FSE theme and the homepage re-adds a title/header, keep the page at its own URL or switch to the free Hello Elementor theme.
