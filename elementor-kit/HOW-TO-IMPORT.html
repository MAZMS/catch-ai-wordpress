Catch AI — WordPress / Elementor Import Guide

Everything to put this page on your own WordPress site. Works on free Elementor — no Pro required.

What you need

  1. A WordPress site (any host).
  2. Elementor (free) — Plugins → Add New → search Elementor → Install → Activate.
  3. The files in this kit (the .json template + the assets/ images).

Import in 5 steps

  1. Import the design: Templates → Saved Templates → Import Templates → upload catch-ai-template-IMPORT.zip (or the catch-ai-elementor-template.json file) → Import Now. Don't upload the whole kit zip — just this template file.
  2. Create a page: Pages → Add New → give it a title → click Edit with Elementor.
  3. Insert the design: in the editor, open the folder / library icon → My Templates → find Catch AI — Home → Insert.
  4. Set full-width layout: Page Settings (gear, bottom-left) → Page Layout → Elementor Canvas → Publish. (This removes the theme header/footer — the design ships its own.)
  5. Set the global colors & fonts: Elementor → Site Settings → Global Colors / Global Fonts using the table below.
No extra CSS setup: the sticky header and the hero's layered phone/photo are driven by a small CSS block that travels inside the template (an HTML widget in the hero), so a fresh import carries it automatically.

Images (optional — host them on your site)

  1. Upload the 7 files in assets/ to the Media Library.
  2. Either swap each Image widget's source in Elementor, or bulk find-replace the base URL with a plugin like Better Search Replace. (By default they load from the preview URL and will work immediately.)
